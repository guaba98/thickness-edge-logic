# 📐 Thickness Edge Logic

두께측정기 프로그램 내에서 **RawData로 엣지를 자동 검출하는 로직**을
차트와 슬라이더로 쉽게 확인할 수 있는 웹 페이지입니다 🧪✨

---

## 🌐 사이트 바로가기

👉 [바로가기](https://guaba98.github.io/thickness-edge-logic/)

---

## ✨ 무엇을 볼 수 있나요?

* RawData 전체 차트 확인
* 슬라이더로 파라미터 실시간 조절
* 1차 엣지 검출 과정 확인
* 2차 엣지 최종 판별 과정 확인

---

## ⚙️ 조절 가능한 값

| 항목            |   기본값 | 설명               |
| ------------- | ----: | ---------------- |
| 최소 두께         | 50 um | 엣지 후보로 인정할 최소 두께 |
| Edge Cut      | 10 um | 앞뒤 위치와의 두께 차이 기준 |
| EdgeFindRatio |  0.90 | 2차 엣지 기준값 계산 비율  |

---

## 🔍 엣지는 어떻게 찾나요?

### 1️⃣ 1차 엣지 찾기

100mm부터 930mm까지 1mm 간격으로 데이터를 훑으면서
두께가 충분히 올라가거나 내려가는 지점을 찾습니다.

* 시작 엣지: 두께가 확 올라가기 시작하는 위치
* 종료 엣지: 두께가 확 내려가기 시작하는 위치

---

### 2️⃣ 2차 엣지 찾기

1차 엣지 주변 데이터를 다시 확인해서
더 정확한 최종 엣지를 찾습니다.

기준값은 아래처럼 계산합니다.

```txt
Threshold = 상위 50번째 두께값 × EdgeFindRatio
```

* Threshold를 처음 넘는 위치 → 최종 시작 엣지
* Threshold 아래로 내려가는 위치 → 최종 종료 엣지
