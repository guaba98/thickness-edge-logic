🔵 1차 엣지 검출 순차 스캔 방식

⚙️ 파라미터

mmData 엣지 최소두께
50
um — 이 이상이어야 엣지 후보
mm Data Edge Cut
10
um — 앞뒤 두께 변화량 기준
모든 위치를 처음부터 끝까지 순차 스캔합니다. 각 위치 X에 대해 X−5, X+5 위치의 두께값을 함께 참조하여 조건을 판별합니다.

시작 엣지 X = 214 (두께 72.8 um)

참조
주변 위치 두께 확인
X−5 = 209 → 두께 16.9 um
X+5 = 219 → 두께 137.1 um
조건1
앞쪽과의 두께 차이 > EdgeCut(10)
현재 두께 − X−5 두께 > 10
72.8 − 16.9 = 55.9

✅ 55.9 > 10 통과
조건2
최소두께 초과 + 상승구간
현재 > 최소두께(50) AND 현재 < X+5 두께 → 상승 중
72.8 > 50
72.8 < 137.1

✅ 두 조건 모두 통과 → 시작엣지 확정

종료 엣지 X = 719 (두께 50.3 um)

참조
주변 위치 두께 확인
X−5 = 714 → 두께 135.7 um
X+5 = 724 → 두께 18.6 um
조건1
뒤쪽과의 두께 차이 > EdgeCut(10)
현재 두께 − X+5 두께 > 10
50.3 − 18.6 = 31.7

✅ 31.7 > 10 통과
조건2
최소두께 초과 + 하강구간
현재 > 최소두께(50) AND 현재 < X−5 두께 → 하강 중
50.3 > 50
50.3 < 135.7

✅ 두 조건 모두 통과 → 종료엣지 확정
1차 엣지 구간 상세 (X: 199~241 / X: 710~741)

📊 기준 두께값 산출 (조건 1 기준값)

1
탐색 범위(204~729)의 RawData를 두께값 기준 내림차순 정렬
총 526개 포인트를 정렬 → 최대값 140.3 um 부터 순서대로 나열
2
상위 50번째 두께값 선정
노이즈나 이상값의 영향을 줄이기 위해 최대값이 아닌 50번째 값 사용
50번째 두께값 = 140.0 um
3
EdgeFindRatio 파라미터 적용
통상 0.9 사용 (히든탭에서 수정 가능, 파라미터화)
140.0 × 0.9 = 126.0 um
✅ 기준 두께값(Threshold) = 126.0 um
이 기준값(126 um)은 소재 내부의 충분한 두께 영역에 진입했는지 판별하는 임계값입니다. 상승구간에서는 이 값을 넘는 첫 지점, 하강구간에서는 이 값 아래로 떨어지는 첫 지점이 최종 엣지가 됩니다.

2차 시작 엣지 X = 216 (두께 128.1 um)

대상
상승구간 (X=209~225) 순차 스캔
